    Sistema de Control Vacaional de Colca-Cola

    Este fué el primer proyecto Java que completé de manera autodidacta gracias a Youtube.

    pantalla principal

    Aqui se le solicita al usuario que ingrese su nombre y contraseña para acceder al prograda.
    Además también se puede observar la imagen de la empresa y un boton que da un error si no se llenó algún campo.

    Pantalla de terminos y condiciones

    En esta pantalla, el boton de continuar estará desabilitado hasta que el usuario seleccione la opción de continuar.
    De lo contrario solo estará habilitado el boton de volver, el cual lleva nuevamente a la página de inicio.

    Pagína principal

    En esta pantalla se muestra el nombre de la persona que se solicitó al inicio, además se le solicita los datos de los trabajadores,
    para de esta manera, de una manera prefijada calcular cuánto tiempo de vacaciones le corresponden al trabajador.

    Otras funcionalidades

    El usuario puede elegir el tema que quiere visualizar: por defecto rojo, verde y negro.
